North and Central American Mammals Functional Rarity Read Me

Matthias Grenié
21 février, 2017

mamm_funrar

This repository contains a worked through example of the use of funrarR package in the context of biogeography using a publicly available dataset on North and Central American mammals (Lawing et al. 2016b; Lawing et al. 2016a).

The dataset is available through Dryad there: doi:10.5061/dryad.9t0n8

Dependencies

The analysis strongly depends on several packages:

  • remake to make the analysis portable and reproducible,
  • dplyr for data wrangling,
  • rdryad to download the date from Dryad,
  • rmarkdown to generate the analysis report,
  • vegan to compute null models for site-species matrix,
  • ggplot2 to plot figures in reports,
  • devtools for session info diagnostics,
  • funrar to compute functional rarity indices.

How to run

To run the analysis on your computer, install remake, make the working directory the project directory and run:

remake::make()

You will see the full analysis run on your computer. With remake all the intermediate objects are viewable on a dependency graph:

remake::diagram()
