North and Central American Mammals Functional Rarity Read Me

Matthias Grenié, Pierre Denelle, Caroline Tucker, François Munoz and Cyrille Violle
2017-03-07

Goal

This repository contains a worked through example of the use of funrarR package in the context of biogeography using a publicly available dataset on North and Central American mammals (Lawing et al. 2016b; Lawing et al. 2016a).

The dataset is available through Dryad there: doi:10.5061/dryad.9t0n8.

The repository is self-contained and can be used to automatically download see How to run section and compute the figures shown in the submitted manuscript.

How to run

To run the analysis on your computer, install remake (see Dependencies section), move to the project directory and run:

remake::make()

You will see the full analysis run on your computer, it can take as long as 15 minutes to run. And you will need to be connected to the internet at least to download the primary data files. With remake all the intermediate objects are viewable on a dependency graph:

remake::diagram()

Organization

The repository is organized with the scripts folder which contains R files to run the analysis. Each R file contains functions used in the remake.yml file for each step of the analysis.

The results folder contains the exploratory_analysis.Rmd file which sums up the results and plot graphs based on the computation done when running the scripts.

The data folder is created when downloading the datafiles.

Dependencies

The analysis strongly depends on several packages:

  • remake to make the analysis portable and reproducible,
  • dplyr for data wrangling,
  • rdryad to download the data from Dryad,
  • rmarkdown to generate the analysis report,
  • vegan to compute null models for site-species matrix,
  • ggplot2 to plot figures in reports,
  • devtools for session info diagnostics,
  • funrar to compute functional rarity indices.
